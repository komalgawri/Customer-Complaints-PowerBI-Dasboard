#Customer Complaints Analytics Dashboard (Power BI)

This Power BI dashboard analyzes customer complaints using a simulated dataset for a banking environment. It highlights issue patterns, resolution efficiency, SLA breaches, product-level trends, and repeat complaints to support decision-making in operations, service improvement, and risk functions.

Key Metrics Included: Key Metrics Included ,Total Complaints, Avg Resolution Time (Hours), SLA Breach %, Repeat Complaint %, Product & Region-Based Trends, Issue-Type & Channel Analysis.

Future Enhancements:  Add AI vs Non-AI comparison dashboard,  Monthly trend analysis + forecasting. Multi-page navigation (Risk View, Product View).
